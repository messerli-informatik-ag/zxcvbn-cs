# zxcvbn-cs

The following changes differ from the orginal one (mickford/zxcvbn-cs - https://github.com/mickford/zxcvbn-cs)

- Add keyboard layouts: 
	Swiss German, Swiss French, Italian, German, French
	
- Add password strength color:
	Returns a color representing how good a password is (0 = bad = red, medium = yellow, good = green)
	
- Changed date matcher logics:
	Considers the different date format by country
